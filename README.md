## Description

This is a simple project to demonstrate how to use the NestJS framework to create authentication and authorization for a REST API.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# debug mode
$ pnpm run debug

# production mode
$ pnpm run prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
